# Vichizzy.github.io

Personal portfolio website for Victor Chimaobim Onuh.

Live at: **https://vichizzy.github.io**

## Setup

This is a static site — no build step required.

To deploy on GitHub Pages:
1. Create a repo named `Vichizzy.github.io` on GitHub
2. Push this folder's contents to the `main` branch
3. GitHub Pages will serve it automatically at `https://vichizzy.github.io`

## Structure

```
Vichizzy.github.io/
├── index.html        # Main page (all sections)
├── css/style.css     # All styles
├── js/main.js        # Scroll behaviour, nav, animations
└── assets/
    └── victor.jpg    # Profile photo
```
